# Construction Consultancy Website

A modern and professional construction consultancy website designed to showcase company services,
, analytics, and client contact information.  
Built with **REACT, TAILWIND, and NODEJS**, the website focuses on reliability, clean design, and user engagement.

## Features

- **Home Page:** Clean landing page introducing the consultancy.
- **Services Section:** Lists core services (design, supervision, project management, etc.) with detailed descriptions.
- **Portfolio / Gallery:** Displays completed and ongoing projects with images and captions.
- **Analytics Page:** (Optional) Displays key performance data or client satisfaction reports.
- **Contact Page:** Includes contact form, phone, email, and Google Map embed.
- **Responsive Design:** Fully mobile-friendly and optimized for all devices.
- **SEO Optimized:** Meta tags and semantic HTML improve discoverability.
